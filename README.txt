TP OpenClassRooms : Créez des pages web interactives avec JavaScript


Instructions:

Ecrire une page web affichant une liste de liens, un peu à la manière de sites comme reddit ou Hacker News.


Projet:
C'est une mini-gestion simple d'utilisation d'api Restfull get/post au format json. 
Les liens existants sont chargés à l'ouverture de la page, puis l'utilisateur à la possibilité d'ajouter de nouveaux liens.



liensweb.html:
Page principale de présentation des liens

liensweb.js:
Le fichier liensweb.js initial contient une liste de liens fournie sous la forme d’un tableau d’objet JavaScript.

liensweb.css:
Feuille de style liée à la présentation de la page principale
